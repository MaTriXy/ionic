rucken-ionic





Admin UI for Ionic4 with Angular7+ mobile application based on Rucken template

What is Rucken?

  • Core - Base entities and services to create applications on Angular7+, includes users, groups and permissions solutions.
  • Web - Base ui grids, inputs and more to manage users, groups and permissions.
  • Ionic - Base mobile ui list, inputs and more to manage users, groups and permissions.

Usage

git clone https://github.com/rucken/ionic.git my-app
cd my-app
npm install
npm run start:prod

Compodoc documentations

http://rucken.github.io/ionic/doc/demo - Demo application documentation

http://rucken.github.io/ionic/doc/ionic - components documentation

Quick links

@rucken/core - Core with Admin UI for web application maked on Angular7+.

@rucken/core-nestjs - A simple application demonstrating the basic usage of permissions with NestJS (JWT, Passport, Facebook, Google+, User, Group, Permission).
